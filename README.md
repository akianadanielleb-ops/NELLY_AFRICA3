# 🍲 NELLY AFRICA 🍲

## Présentation

NELLY AFRICA est un mini-projet réalisé dans le cadre du cours **S2 - Fondations HTML & Sémantique Native**.

L'objectif du projet est de concevoir un site web consacré à la cuisine africaine en utilisant exclusivement du HTML sémantique et le fichier de style fourni `semanstyle.css`.

Le site présente plusieurs spécialités africaines, des informations sur le restaurant, ainsi qu'un formulaire de réservation accessible.

---

## 🎯 Objectifs du projet

* Comprendre la structure d'une page web HTML.
* Utiliser les balises sémantiques correctement.
* Créer un formulaire accessible.
* Respecter les standards du Web.
* Valider le code avec le validateur W3C.

---

## 🍛 Plats présentés

* Saka_Saka_et_Foufou
* Tripes_Sautées
* Riz_au_Poisson

---

## 🏗️ Structure HTML utilisée

Le projet utilise les balises sémantiques suivantes :

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<article>`
* `<aside>`
* `<form>`
* `<fieldset>`
* `<legend>`
* `<footer>`

---

## ♿ Accessibilité

Chaque champ du formulaire est associé à un label.

Le formulaire utilise :

* `label`
* `input`
* `select`
* `textarea`
* `fieldset`
* `legend`

afin de respecter les bonnes pratiques d'accessibilité.
